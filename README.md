# SIHD

Simple Input Handler and Displayer

## Usage

Just put it in: $HOME/.local/lib/python-X.Y/site-packages/.

Or in your $HOME/.bashrc: export PYTHON_PATH="$PYTHON_PATH:/path/to/sihd"

Or pip3 install -e /path/to/sihd
